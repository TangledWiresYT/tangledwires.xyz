Random Announcements

Make a list of announcements and one of them will be randomly chosen every 5 minutes and broadcast to the server.

Commands:

  • /announcements help - will show a help page on how to use this script
  • /announcements list - will show a list of all announcements
  • /announcements add <text> - adds the message to the announcement list
  • /announcements remove <number> - removes the message with the specified id from the announcement list
  • /announcements disable - disables the announcements
  • /announcements enable - enables the announcements

    • You can edit other options from within the script.

    Addons Required:

  • SkBee
